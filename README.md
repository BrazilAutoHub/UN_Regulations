## 📄 文档类型

每个法规提供三种文档格式：

### 1. HTML 介绍页面
- **用途**：快速了解法规基本信息
- **内容**：法规概述、适用范围、核心内容、技术要求摘要
- **特点**：响应式设计，支持移动端访问
- **文件命名**：`R11_03.html`、`R12_04.html` 等

### 2. Word 摘要文档
- **用途**：离线阅读和打印
- **内容**：法规框架、核心章节摘要
- **特点**：专业报告格式，包含封面和目录
- **文件命名**：`R11_03.docx`、`R12_04.docx` 等

### 3. 全文翻译 Word 文档
- **用途**：深入研究和技术应用
- **内容**：法规原文完整翻译，包含所有章节和附录
- **特点**：保留原文章节结构，专业术语准确
- **文件命名**：`R11_03_full.docx`、`R12_04_full.docx` 等

## 📖 法规分类

### 被动安全
- **R11**：门锁和车门保持件 - 防止碰撞时车门开启
- **R12**：转向机构碰撞保护 - 保护驾驶员免受转向系统伤害
- **R14**：安全带固定点 - 安全带和儿童座椅安装基础
- **R16**：安全带及约束系统 - 乘员约束装置
- **R17**：座椅、座椅固定点和头枕 - 座椅强度和头枕保护
- **R21**：车辆内部安装件 - 内饰安全设计

### 主动安全
- **R13H**：乘用车制动系统 - 制动性能要求
- **R139**：电子稳定性控制系统（ESC）- 防止车辆失控
- **R140**：轮胎压力监测系统（TPMS）- 预防爆胎事故
